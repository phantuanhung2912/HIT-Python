Bài 2
- Các kiểu dữ liệu trong Python:
+ int (integer): Số nguyên
+ float (floating - point Number): số thực
+ complex (complex number): số phức
+ str (string): chuỗi kí tự
+ list (list): danh sách
+ tuple: bộ dữ liệu
+ set: tập hợp
+ dict (dictionary): từ điển
+ none: giá trị rỗng

2. Các toán tử trong Python
a) Toán tử số học
+ : Cộng
- : Trừ
* : Nhân
/ : Chia
// : Chia lấy phần nguyên
% : Chia lấy dư
** : Lũy thừa
b) Toán tử so sánh
== : Bằng
!= : Khác
> : Lớn hơn
< : Nhỏ hơn
>= : Lớn hơn hoặc bằng
<= : Nhỏ hơn hoặc bằng
c) Toán tử gán
=
+=
-=
*=
/=
//=
%=
**=
d) Toán tử logic
and
or
not
e) Toán tử thành viên
in
not in
f) Toán tử định danh
is
is not

- Mệnh đề điều kiện
